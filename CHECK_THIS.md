## 1. Открой эти ссылки в браузере:

### Тест Backend API:
http://localhost:3001/api/health

**Должно быть:**

```json
{"status":"ok","timestamp":"2026-01-24T..."}
```

### Главный сайт:
http://localhost:5173

**Должно быть:** Красивая страница CaseFun с заголовком "Your Token, Your Cases, Your Rules."

## 2. Если сайт НЕ открывается:

### Шаг 1: Проверь что серверы запущены
В терминале должно быть что-то вроде:
```
> frontend@0.0.0 dev
> vite

  VITE v6.x.x  ready in xxx ms

  ➜  Local:   http://localhost:5173/
  ➜  Network: use --host to expose
```

### Шаг 2: Если не видишь этого - запусти:
```bash
cd /Users/satory/Downloads/casefun
npm run dev
```

## 3. Если сайт открывается но есть ошибки:
